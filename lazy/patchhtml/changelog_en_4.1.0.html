4.1.0 Important Enhancements (9/16/2022)

Spellextensions functional

Spell extensions can now have active effects and modify the attributes of spells dynamically. You can now create extensions which increase e.g. the damage, range or anything else.
All spell attributes should work as modifier. Since TDE has more than 3000 extensions it is unlikely that we add all those mods ourselves. But you can contribute on a voluntary basis!

{
    {"key": "system.effectFormula.value", "mode": 2, "value": "+2"}, 
    {"key": "system.range.value", "mode": 5, "value": "32 Schritt"}, 
}

A complete coverage or functionality is only possible through joined efforts by the community of TDE/DSA. Feel free to contribute, we'll add the effects in intervals to the official modules. See https://github.com/Plushtoast/dsa5-extensiondb

Area templates for spells

Spells have now settings for area templates at the details tab. After a successfull cast you can drop the template to the canvas for easy measurement. The distance values support formulas which include the QS/QL of the roll, e.g. "qs*2". We will see if we can improve on this in the future to apply spell effects (with animations) and similar stuff.

Info item for plants & journals

Text areas can now include a preview for information items which contain the roll request and all required information.
To create one drag an information item to the textarea and replace UUID with Info like so:


        @Info[Item.3XAB0PU0Z6IS9YA]

Plants respectively the Herbarium will get this by default soon. Adventures will be updated over time.

Obfuscate items

The gamemaster has now small buttons on itemsheets to hide information on a single tab to the players. This way you can make mysteriös items with unknown effects or prevent metagaming. The herbarium will be updated to allow for default occlusion of critical information by default.

Enhancements

  • Maintained costs of spells are now added to the spell costs and become an additional effect on the casting actor which deducts AE automatically in intervals until deleted (thanks Qowy, requires times-up module)
  • Creatures can now use ammunition too (but they still don't have to)
  • Shapeshifting can now keep gear and token art (for possesion)
  • NPC and character sheets can now also set the creature size
  • The library brain filter can now search for combatskills in combat special abilities
  • Tradition artifacts can now be tracked better
  • Merchants have now a + button to select actors tracked by the game master menu (table hybrid)
  • The token hotbar has been altered
  • The english core module receives an additional hires aventuria map
  • The orcs module receives the new race holberker

Bugfixes

  • Token obfuscation fixed
  • Fixed a bug showing different speed values to players and gm when token affected by pain
